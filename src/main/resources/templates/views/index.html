Welcome to GateWay

GateWay provides flight reservation and checkin services.

Pre-loaded Users for Login:

Email Password Authentication
admin@admin.com admin ADMIN, USER
user@user.com user USER

Functionalities:

  • Support user registration
  • Support guest search (or condition) and check all flights (blank condition)
  • After registered user submit the flight reservation, a digital itinerary will be sent to Passenger's email address
  • Registered users should use flight reservation ID to check in and update number of bags
  • Login as admin to access flight create function in "Add Flights" tag, flight edit and delete function in "Flight Search" tag

Tech Stack:

Spring Boot App, Spring Data (JPA, Hibernate, ORM), Spring MVC, Spring Security, Thymeleaf template engine dialects, RESTful web service, SLF4j/Log4j logging, Exception handling, Maven builder, Git, Heroku deployment, JawsDB MySQL on Heroku, HTML/CSS/JavaScript/BootStrap
